# @dizmo/functions
Library of useful functions, which can be used in developing dizmos.

## Usage
### Install
```sh
npm install @dizmo/functions --save
```
### Require
```javascript
var lib = require('@dizmo/functions');
```
### Examples
```javascript
var buffered = require('@dizmo/functions').buffered;
```

## Development
### Build
```sh
npm run build
```
### Lint
```sh
npm run lint
```
#### with fixes:
```sh
npm run lint:fix
```
### Test
```sh
npm run test
```
### Cover
```sh
npm run cover
```

# DizmoFunction
Library of useful functions, which can be used in dizmos.

## Installation 
```sh
npm install @dizmo/dizmo-functions --save
```

## Usage
```javascript
var functions = require('@dizmo/dizmo-functions');
```

## Test 
```sh
npm run test
```

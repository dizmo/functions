# DizmoFunctions
Library of useful functions, which can be used in developing dizmos.

## Installation
```sh
npm install @dizmo/dizmo-functions --save
```

## Usage
```javascript
var fn = require('@dizmo/dizmo-functions');
```

## Build
```sh
npm run build
```

## Lint
```sh
npm run lint
```
### with fixes:
```sh
npm run lint:fix
```

## Test
```sh
npm run test
```

## Cover
```sh
npm run cover
```

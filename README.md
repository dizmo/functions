# DizmoFunction
Library of useful functions, which can be used in developing dizmos.

## Installation 
```sh
npm install @dizmo/dizmo-function --save
```

## Usage
```javascript
var fn = require('@dizmo/dizmo-function');
```

## Test 
```sh
npm run test
```

# @dizmo/functions
Library of useful functions, which can be used in developing dizmos.

## Usage
### Install
```sh
npm install @dizmo/functions --save
```
### Require
```javascript
var fn = require('@dizmo/functions');
```
### Examples
```javascript
var buffered = fn.buffered;
```
```javascript
var random = fn.random;
```

## Development
### Build
```sh
npm run build
```
### Lint
```sh
npm run lint
```
### Test
```sh
npm run test
```
### Cover
```sh
npm run cover
```
